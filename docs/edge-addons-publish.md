## Edge Addons API workflow

Get `EDGE_CLIENT_ID` from the [Partner Center](https://partner.microsoft.com/en-us/dashboard/microsoftedge/overview).

Get the `EDGE_API_KEY` from [Publish API](https://partner.microsoft.com/en-us/dashboard/microsoftedge/publishapi).

> APIKey can expired, to renew it go to the Publish API section and update the environment variable.

[Documentation on Rest API](https://learn.microsoft.com/en-us/microsoft-edge/extensions/update/api/using-addons-api?tabs=v1-1#enable-the-update-rest-api-at-partner-center)

Support on [MicrosoftEdge-Extensions GitHub](https://github.com/microsoft/MicrosoftEdge-Extensions?tab=readme-ov-file)
